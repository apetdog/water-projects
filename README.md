# Water Projects Launcher

## 安装依赖

首先需要安装 nodejs 20 以上版本，以及安装 pnpm 包管理器。

可以从这里安装 nodejs：[Node.js](https://nodejs.org/en/download/)

可以使用以下命令安装 pnpm：

```bash
npm install -g pnpm
```

接下来需要在项目根目录安装依赖：

```bash
npm install
```

## 启动项目

可以使用以下命令启动所有项目：

```bash
npm run dev
```
项目会依次启动，每个项目会在不同的端口运行。

01a-water-twin-screen 对应的地址为 http://127.0.0.1:3000/water-twin-screen/
