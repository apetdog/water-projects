# Hacker Typer (终端极速编码模拟器)

这是一个用来"假装"你正在以极快速度编写复杂代码的脚本。无论你在键盘上输入什么内容，屏幕上都会显示预设的硬核内核代码。

## 版本说明

两个不同风格的脚本，满足不同的表演需求。

### v1: 极速编码模式 (`coding_hacker.sh`)

**适合场景**：假装自己在疯狂敲代码，手速惊人。

- **玩法**：运行脚本后，在键盘上**随意乱按**任意键。
- **效果**：屏幕上会根据你的按键节奏，逐字逐句地输出一段彩色的 Linux 内核 C 代码，带有语法高亮。
- **亮点**：支持关键字高亮（如 `void` 是黄色，`#include` 是青色），看起来非常专业。

```bash
chmod +x coding_hacker.sh
./coding_hacker.sh

```

结束脚本演示：按下`ctrl+c`，脚本会安全退出。

### v2: 命令行黑客模式 (`debug_hacker.sh`)

**适合场景**：假装自己在入侵系统，输入指令后系统疯狂反馈。

- **玩法**：
  1. 看到 `root@darknet-core:~#` 提示符后，一本正经地输入一行命令（比如 `hack satellite --target=nsa` 或者随便乱打）。
  2. 潇洒地按下 **回车 (Enter)**。
  3. 屏幕上会自动滚出一大段看起来很厉害的日志（如 Nmap 扫描结果、Metasploit 攻击过程、内存 Hex 导出等）。
  4. 结束后会再次出现提示符，重复上述过程。
- **效果**：模拟真实的终端交互，输出内容包含多种颜色和格式。

```bash
chmod +x debug_hacker.sh
./debug_hacker.sh
```

结束脚本演示：按下`ctrl+c`，脚本会安全退出。

## 技巧

- **v1**: 保持敲击的节奏感，时快时慢会让表演更真实。最后可以猛敲一下回车，配合脚本结束特效。
- **v2**: 在按下回车前，可以假装思考一下，或者跟旁边的人解释你在"配置参数"。
