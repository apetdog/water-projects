# 智慧管控平台 - 需求梳理文档

## 1. 项目概况
本项目旨在构建一个智慧园区管控平台的可视化大屏页面。项目结合了 3D 园区模型、实时视频监控（海康 SDK）、环境监测数据展示以及模拟报警交互功能。

## 2. 核心功能模块

### 2.1 导航与页面结构
系统主要包含四个一级导航/标签页：
1.  **监控画面**
2.  **安防系统**
3.  **LED 大屏**
4.  **环境检测**

> **视觉增强要求**：对于内容较为空泛的标签页，需增加 3D 模型互动或高视觉表现力的动态效果（"表演性强的玩意"）。

### 2.2 视频监控模块
- **核心要求**：必须接入真实的海康威视 WebSDK（无插件版）。
- **功能**：支持实时画面预览。
- **参考资源**：
    - SDK 包：`WebSDK_noPlugin_V3.4.0`
    - 参考图：`海康监控画面.jpg`

### 2.3 3D 园区模型与无人机视角
- **模型要求**：
    - 使用外观精美的通用园区 3D 模型。
- **无人机视角**：
    - **功能**：在模型展示中增加无人机俯瞰或巡航视角。
    - **交互**：支持**点击放大**，交互体验需与监控画面保持一致（即支持弹窗或全屏查看）。
    - **参考图**：`无人机视角.jpg`
    - **相关素材**：目录下存在 `5cade2f70e5abe8cbcbe201c60344151.mp4`，可能作为无人机视角的视频素材或参考。

### 2.4 报警模拟功能
- **触发机制**：在页面角落设置一个报警按钮。
- **交互逻辑**：点击按钮后，弹出一个报警详情界面。
- **数据来源**：使用静态数据或假数据进行模拟展示。
- **参考图**：`报警.jpg`

### 2.5 环境监测
- **功能**：展示园区环境相关数据（如温湿度、PM2.5 等）。
- **参考图**：`环境检测.jpg`
