# Shield Scan

这是一个炫酷的终端演示脚本，模拟了多种网络安全和系统监控场景。通过使用 ASCII 艺术和 ANSI 颜色代码，在终端中展示生动的动画效果。

## 功能特性

脚本包含以下模拟场景：

- **系统安全扫描**：模拟文件完整性扫描。
- **网络流量监控**：展示实时网络连接和流量分析。
- **入侵检测系统**：模拟检测 SQL 注入、XSS、DDoS 等攻击。
- **防火墙与病毒库更新**：动态展示规则更新过程。
- **安全连接建立**：模拟加密通道握手流程。
- **系统资源监控**：实时 CPU、内存、磁盘监控面板。
- **全球攻击来源**：可视化的世界地图攻击源分布。
- **实时数据图表**：动态波形图和统计柱状图。
- **雷达扫描系统**：模拟雷达探测目标。
- **黑客帝国矩阵雨**：经典的绿色代码雨特效。

## 使用方法

在终端中运行以下命令启动演示：

```bash
chmod +x ShieldScan.sh
./ShieldScan.sh
```

### 命令行参数

脚本支持以下参数配置：

- `-t, --time <seconds>`: 设置演示运行时长（单位：秒）。默认值为 600 秒（10 分钟）。
- `-h, --help`: 显示帮助信息。

**示例：**

```bash
# 运行默认时长（10分钟）
./ShieldScan.sh

# 运行 5 分钟 (300秒)
./ShieldScan.sh -t 300

# 运行 1 分钟并显示帮助
./ShieldScan.sh --time 60
```

## Windows 用户说明

**Windows 上建议下载一个 Git**

由于这是一个 Bash 脚本，Windows 用户需要一个类 Unix 的 Shell 环境来运行它。推荐安装 [Git for Windows](https://git-scm.com/download/win)，安装后使用 **Git Bash** 来运行此脚本。

1. 下载并安装 Git for Windows。
2.而在文件夹空白处右键，选择 "Open Git Bash here"。
3. 输入 `./ShieldScan.sh` 运行。
